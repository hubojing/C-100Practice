题目：有5个人坐在一起，问第五个人多少岁？他说比第4个人大2岁。问第4个人岁数，他说比第3个人大2岁。问第三个人，又说比第2人大两岁。问第2个人，说比第一个人大两岁。最后问第一个人，他说是10岁。请问第五个人多大？
```c
#include<stdio.h>

int f(int n)
{
	int age = 0;
	if (n == 1)
	{
		return 10;
	}
	else
	{
		age = f(n - 1) + 2;
		return age;
	}
}

int main()
{
	int age = f(5);
	printf("%d", age);
	return 0;
}
```
